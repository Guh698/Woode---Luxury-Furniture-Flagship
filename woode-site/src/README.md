# Woode

![Woode Flagship Experience](https://res.cloudinary.com/dabshzrnj/image/upload/v1773965626/woodeHero_khihtc.webp)

> _Digital experience for a conceptual luxury furniture brand._

### Overview

Developed by Studio JEMD, this repository houses the front-end architecture for the Woode flagship web experience. The interface is crafted to reflect the brand's high-end, architectural nature, prioritizing fluid motion, typographic precision, and seamless content delivery.

By leveraging a headless CMS alongside fluid page transitions, the platform removes non-essential friction, allowing the conceptual furniture pieces to take center stage in a refined, editorial format.

### Technical Architecture

- **Frontend & Tooling:** Vanilla JavaScript (ES6+) and Vite
- **Content Management:** Sanity (Headless CMS)
- **Page Transitions:** Barba.js
- **Motion Engine:** GSAP
- **Styling:** CSS3 Custom Properties
